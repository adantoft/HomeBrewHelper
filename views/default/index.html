HomebrewHelper is designed to help the homebrewer keep track of the beers that have been brewed, monitor fermentation temperatures, and keep an eye on how much beer is remaining in a keg. The original goal of this project was to experiment with Data-Driven Documents (D3), a graphing Javascript library, and its integration with Web2Py. During the coding of this project, it became obvious that the flexibility with D3 is nearly limitless. The D3 functionality of this site could be fine-tuned and turned into a plugin. The flexible nature of D3 does however make easy adaptability for a plugin challenging.
D3 examples from the web were used as starting points for the D3 functionality in this site. References are included in the source code.
